# 中国日历

> 红过节，黑上班

一眼看清哪天放假、哪天补班的农历日历 App。

---

## 颜色规则

| 颜色    | 含义              |
| ------- | ----------------- |
| 🔴 红色 | 法定节假日 / 周末 |
| ⚫ 黑色 | 调休补班          |
| 🔵 蓝色 | 今天              |
| 灰色    | 非当月日期        |

---

## 功能

- 农历、干支、生肖、节气同步显示
- 节假日数据实时从接口获取，无需手动更新
- 点击任意日期查看详细农历信息

---

## 技术栈

- React Native / Expo
- date-fns · lunar-javascript
- [jiejiariapi.com](https://api.jiejiariapi.com) 节假日接口
